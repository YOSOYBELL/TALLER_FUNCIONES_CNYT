# Librería Operaciones con Vectores y Matrices Complejas

En este proyecto se implementan las principales operaciones con vectores y matrices complejas utilizando Python y la librería NumPy.

El cuaderno de Jupyter desarrolla paso a paso las siguientes operaciones:

- Adición de vectores complejos  
- Inverso aditivo de un vector  
- Multiplicación de un escalar por un vector  
- Adición de matrices complejas  
- Inverso aditivo de una matriz  
- Multiplicación de un escalar por una matriz  
- Transpuesta de una matriz o vector  
- Conjugada de una matriz o vector  
- Adjunta (daga) de una matriz  
- Producto de matrices  
- Acción de una matriz sobre un vector  
- Producto interno entre vectores  
- Norma de un vector  
- Distancia entre vectores  
- Cálculo de valores y vectores propios  
- Verificación de matriz unitaria  
- Verificación de matriz Hermitiana  
- Producto tensorial  

Todas las operaciones se implementan utilizando arreglos (`ndarray`) de la librería NumPy.



## Pre-requisitos

Para ejecutar este proyecto necesitarás:

- Python 3.x  
- NumPy  

Puedes instalar NumPy con el siguiente comando:


pip install numpy
